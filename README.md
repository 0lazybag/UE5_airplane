UE5 C++ Systems: Transport, NPC & Interaction Logic

Репозиторий содержит демонстрацию архитектурного подхода к созданию игровых систем на Unreal Engine 5 с использованием чистого C++. Проект был реализован за 8 рабочих дней в качестве технического кейса.

📂 Обзор архитектуры и классов

Проект демонстрирует реализацию различных систем управления и ИИ, от простых патрульных ботов до сложных физических моделей транспорта.
🎮 Игрок и системы взаимодействия

    CPP_Character1.cpp: Ядро игрока. Реализует расширенную логику перемещения (спринт, прыжки) и централизованную систему взаимодействия. Использует LineTrace для обнаружения NPC различных типов (CPP_NPC01-05) и управляет их состоянием (заморозка движения, поворот к игроку, вызов диалоговых виджетов).

🤖 Системы ИИ (NPC)

Каждый NPC демонстрирует разные подходы к реализации автономного поведения:

    CPP_NPC01.cpp: Линейный патруль. Реализация циклического движения между точками с использованием интерполяции и состояний ожидания.

    CPP_NPC02.cpp: Реактивный ИИ. Бот, меняющий направление при столкновениях (OnHit), имитирующий хаотичное движение в замкнутом пространстве.

    CPP_NPC03.cpp: Умный обход препятствий. Использует периодические проверки (LineTraceSingleByChannel) для обнаружения преград и плавного восстановления маршрута.

    CPP_NPC04.cpp: Продвинутый навигатор. Использует многолучевое сканирование (Sensing) для оценки весов направлений и выбора оптимального пути в реальном времени.

    CPP_NPC05.cpp: Путевые точки (Waypoints). Система следования по заданному маршруту с логикой избегания столкновений и динамическим разворотом к цели.

✈️ Транспортные системы

    airplane_cpp.cpp: Сложная физическая модель самолета. Управление тягой левого/правого двигателей через карты мощностей (PowerMap), расчет углов тангажа (Pitch) и крена (Roll), обработка состояния «на земле» и торможения.

    airplane2.cpp: Математическая модель движения по траектории. Реализация полета по сложным кривым с использованием тригонометрических функций (Sin) для вычисления координат.

    Car01.cpp: Логика наземного транспорта. Реализует движение по маршруту с остановками, автоматическую анимацию вращения колес (WheelSpinAxis) в зависимости от пройденного пути и анимацию дверей.
